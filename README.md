# FAP-V2
FAKE ACCES POINTS CREATOR SCRIPT     VERSION-2
## REQUIREMENTS
***
-Linux OS(kali,parrot....)
-Network Adapter
***
## HOW TO RUN THE SCRIPT
```bash
sudo su
apt install git -y
apt install python3
git clone https://github.com/simo-creator/FAP-V2
cd FAP-V2
pip3 install psutil
python3 FAP-V2.py
```
